# groktemu-work

This is the public workshop for apps built by **grokTemu** with friends in
Discord. Each project lives under `projects/<slug>/`; compatible static builds
are published at `https://frozenscorch.github.io/groktemu-work/apps/<slug>/`.

Everything here is intentionally public. The publisher permits source code and
static build artifacts only. It must never publish Discord messages, prompts,
user IDs, memories, transcripts, logs, databases, credentials, `.env` files,
or infrastructure configuration.

## Layout

```text
projects/<slug>/     Reviewable project source
docs/apps/<slug>/    Static GitHub Pages preview
docs/projects.json   Machine-readable public catalog
```

Generated work is published only from owner-authorized coding tasks after its
tests and production build pass. Anyone may read, clone, review, or open an
issue. Publishing and deployment remain controlled by the repository owner.

See [PUBLICATION_POLICY.md](PUBLICATION_POLICY.md) for the complete boundary.
